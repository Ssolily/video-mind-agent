# Demo Walkthrough

## 推荐演示流程（5 分钟）

### 1. 启动项目

```powershell
# 终端 1：启动后端
cd backend
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000

# 终端 2：启动前端
cd frontend
npm run dev
```

### 2. 上传视频

- 打开浏览器访问 `http://localhost:5173`
- 首页显示产品介绍和开始分析入口
- 点击上传区域或拖拽 mp4/mov/avi 文件
- 显示文件大小和格式提示
- 设置分析目标和参数

### 3. 查看任务进度

- 上传后自动进入分析状态
- 任务状态卡片显示进度、当前步骤
- 支持取消和重试操作
- 任务状态：排队中 → 分析中 → 完成/失败

### 4. 打开结果页

- 分析完成后自动显示结果
- 结果布局：视频播放器 + Highlight 列表 + Timeline
- 支持播放、跳转、导出片段

### 5. 点击 Timeline highlight

- 时间轴上显示所有精彩片段的区间
- 点击片段跳转到对应时间位置
- 当前播放位置在时间轴上有游标
- 悬浮显示起止时间和评分

### 6. 查看 InsightPanel 和报告

- 展开报告面板查看分析摘要
- 复制摘要和时间段到剪贴板
- 打开 Markdown 报告链接
- 打印报告

### 7. 复制和操作反馈

- 复制摘要/时间段/JSON 时有 Toast 通知
- 操作成功/失败均有反馈提示
- 错误状态显示友好文案

### 8. 打开历史页

- 在导航栏点击 "历史记录"
- 查看所有历史分析任务
- 支持搜索、筛选、排序
- 点击任务再次查看结果

### 9. 主题切换

- 支持 light / dark / system 三种主题
- 导航栏右侧的主题切换按钮
- 所有组件适配暗色模式

## 主题说明

- 默认跟随系统偏好 (prefers-color-scheme)
- 手动切换后保存到 localStorage
- 确保所有组件在 light/dark 下都可读

## 本地历史说明

- 历史记录仅保存在浏览器 localStorage 中
- "清空本地历史"不会删除服务器上的视频或分析结果
- 清除后无法恢复

## 已知限制

1. 需要本地运行后端和前端
2. 首次分析需要加载模型
3. GPU 模式下效果更佳
4. 长视频分析时间较长
